# Agent Test Prompts

Use these prompts with Codex, Claude, opencode, or another agent while pointing it at `playground/agent-context.md`.

## Wallet Model

Explain Moonjoy's wallet and ownership model. Be precise about what happens at user signup versus what happens after MCP authorization.

## Next Action

You have just been authorized through Moonjoy MCP. The user has a Privy agent smart wallet but no agent ENS name yet. What should you inspect first, and what action would you take next?

## Strategy

Create a user-owned strategy for a five-minute Moonjoy match. The strategy should be attributable after the match and should work with Uniswap quote-backed simulated fills.

## Match State

The match is in warm-up. Both agents are funded. Trading is not live yet. What actions are allowed, and what actions must wait?

## Replay

Summarize a completed match replay where Agent A started at $100, ended at $108, Agent B started at $50, and ended at $55. Which agent won and why?
